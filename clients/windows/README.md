# X87 Player - Windows Client

Windows desktop IPTV player application built with Python + PyQt6 + MPV (embedded) + VLC (external launch).

## Requirements
- Python 3.9+
- PyQt6
- python-mpv (requires `libmpv`)
- python-vlc (for "Open in VLC" external playback)
- requests
